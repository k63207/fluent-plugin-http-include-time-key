## Installation

### RubyGems

```
$ gem install fluent-plugin-http-include-time-key
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-http-include-time-key"
```

And then execute:

```
$ bundle
```

```
$ fluent-plugin-config-format input http_include_time_key
```
